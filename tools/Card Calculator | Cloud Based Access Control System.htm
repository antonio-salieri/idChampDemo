The Brivo Card Calculator translates hexadecimal values into readable card format information, providing the internal card number, facility code, and card format.
The hex values (for non-proprietary card formats) that can be calculated are Standard 26 Bit, Generic 33 Bit, 34 Bit (even or odd parity), HID Corporate 1000 35 Bit, 37 Bit (with or without facility code), and Casi-Rusco 40 Bit.
To utilize the card calculator, select the appropriate bit length from the dropdown menu under Input in the card calculator. You can find the bit length in the activity log of Brivo OnAir along with the unknown hex value. Next, copy that hex value from the Brivo OnAir activity log page and paste it into the Hex field under Input in the card calculator. Click on the Calculate button and the hex value is translated into recognizable card format information.
For more information, please refer to the Brivo OnAir Administrator’s Manual.
